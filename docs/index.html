Budgie

A plugin to handle all your scrolling needs.

Horizontal scroll, with autoScroll on.
var exampleOne = new InfiniteScroller( sampleImages, '#example-1', { 'numberHigh': 1, 'numberWide': 4, 'secondsOnPage': 15.0, 'direction': 'horizontal', 'autoScroll': true } );
Inverted Horizontal scroll, with autoScroll on.
var exampleTwo = new InfiniteScroller( sampleImages, '#example-2', { 'numberHigh': 1, 'numberWide': 4, 'secondsOnPage': 15.0, 'direction': 'horizontal', 'inverted': true, 'autoScroll': true } );
Inverted vertical scroll, with autoScroll on.
var exampleThree = new InfiniteScroller( sampleImages, '#example-3', { 'numberHigh': 2, 'numberWide': 3, 'secondsOnPage': 15.0, 'direction': 'vertical', 'inverted': true, 'autoScroll': true } );
